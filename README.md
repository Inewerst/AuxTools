## 问题列举
1. 在虚拟机里运行, 程序空白怎么解决?   
1) 如果是VMware, 打开VM虚拟机->首选项->显示->图形->勾选远程虚拟机的硬件加速 选项;如果打开还是白屏, 编辑虚拟机设置->显示器->3D图形->取消勾选3D加速图形
2) 如果是VirtualBox, 打开设置->显示->显卡->勾选3D加速选项
